План тестирования

Будем тестировать:
	В файле test_routes.py:
	 v- Страницы home, login, logout, signup доступны анонимному пользователю.
	 v- При попытке перейти на страницу add, edit, detail, delete, list, success аноним перенаправляется на страницу авторизации.
	 v- Страницы add, edit, detail, delete, list, success доступны авторизованным пользователям.
	 v- Авторизованный пользователь не может редактировать просматривать или удалять чужие записи (возвращается ошибка 404).

	В файле test_content.py:
	 v- Записи отсортированы по возростанию id.
	 v- При добавлении и изменении записи на странице доступна форма класса NoteForm

	В файле test_logic.py:
	 v- Анонимный пользователь не может отправить запись.
	 v- Авторизованный пользователь может отправить запись.
	 v- Ошибка при создании неуникального слага записи
	 v- Авторизованный пользователь может редактировать или удалять свои записи.
	 v- Авторизованный пользователь не может редактировать или удалять чужие записи.
